** What is Bootstrap?

	It's the most populat HTML,CSS and JS framework for developing responsive,
	mobile first projects on the web.



	* How to include Bootstrap using download?

	download bootstrap and copy the bootstrap.css and move it into your folder, and include just like a normal css.

	
	* How to include Bootstrap using just the link?
	
	Go to Bootstrap CDN, and copy the latest compiled and minified CSS and paste into your html file.

	** Important **
		It's important to get the most out of Bootstrap, you should also include the javascript and jquery at the bottom of body tag

		it should look somthing like this:

		
		<script src="https://code.jquery.com/jquery-3.2.1.js" integrity="sha256-DZAnKJ/6XZ9si04Hgrsxu/8s717jcIzLy3oi35EouyE=" crossorigin="anonymous"></script>

		<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous"></script>

	** Responsive for mobile **
		If you want your bootstrap styled website to be responsive on mobile then be sure to add 
		the following meta tag to your <head>  element, above the <title>  tag:
		
		<meta name="viewport" content="width=device-width, initial-scale=1">
