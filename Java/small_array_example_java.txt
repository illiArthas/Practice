** Java Lynda Notes **

	To declare an Array of 5 integers named nigga:
		int[] nigga = new int[5];

	To create a Random object name r, that generates a random number 
	between 1 and 100:
		int num = r.nextInt(100) + 1;

	To declare a two-dimensional string array with 5 rows and 10 columns:
		String[][] aString = new String[5][10];
