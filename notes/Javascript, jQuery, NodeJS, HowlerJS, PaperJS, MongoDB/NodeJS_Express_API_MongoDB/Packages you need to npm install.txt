Notes for each steps

1. Express - 1, 2, 2.5

2. EJS - Express_Template and EJS_3.txt (EJS is a template, kind of like .html)

3. Body-parser - Express_post_put_delete_requests_4.txt (Body-parser is for parsing the post data)

4. API folder, then open the text file

5. MongoDB folder, then open the text file (Mongoose is for the database)

6. Express folder, then open Express_post_put_delete_requests_4.txt
